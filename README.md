# ADVENTURE-WORK-DASHBOARD
Project Overview
This project provides a comprehensive data analysis and visualization of the AdventureWorks ecosystem. Utilizing a structured relational data model, the project transforms raw transactional data into actionable business insights focused on global sales performance, regional trends, and customer demographics.

Data Architecture
The analysis is built upon a robust schema consisting of several key dimensions and fact tables:
Sales Performance (FULL_SALES): Centralized transactional data including order quantities, unit prices, and total revenue.  
Product Intelligence (Final_product_data): Detailed cataloging of products to track performance by line and category.  
Customer Demographics (DimCustomer): Profiles to identify high-value segments and purchasing behaviors. 
Temporal Analysis (DimDate): A dedicated time dimension for year-over-year and month-over-month growth tracking.  
Geographic Insights (DimSalesTerritory): Mapping sales performance across global territories.

Key Analytics & Visualizations Time-Series Tracking:

Year-wise, quarter-wise, and month-wise sales breakdowns to identify seasonal trends and growth patterns.  
Interactive Dashboards:
Advanced use of pivot tables and combo charts to visualize the relationship between different metrics. 
Operational KPIs: Integrated Key Performance Indicators (KPIs) and dynamic slicers for real-time data filtering and executive reporting. 

Technical Stack
Data Processing: Excel/CSV for structured data management.  
Visualization: Tableau (Packaged Workbook) for interactive dashboarding.  
Analytics: Advanced Pivot Analysis and KPI modeling.
